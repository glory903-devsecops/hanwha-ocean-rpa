# [BRD] Business Requirements Document: Hanwha Ocean Smart Yard AX

## 1. 프로젝트 배경 (Background)
한화오션은 글로벌 해양 솔루션 리더로서, 생산 혁신을 위해 **스마트 야드(Smart Yard)** 구축 및 **AX(AI Transformation)**를 추진하고 있습니다. 거대한 야드 환경에서 산재된 공정 데이터를 수동으로 관리하는 비효율을 제거하고, 데이터 기반의 실시간 의사결정 체계를 구축하는 것이 본 프로젝트의 핵심 목적입니다.

## 2. 비즈니스 목표 (Business Objectives)
- **조업 효율성 향상**: 도크별 실시간 공정률을 자동 취합하여 전체 평균 공정률을 지표화함.
- **안전 관리 고도화**: 휴먼 에러를 배제한 실시간 안전 이슈 모니터링 및 교육 이수율 관리.
- **AX 가치 증명**: AI 기반의 공정 예측(완공 예정일) 기능을 통해 단순 모니터링을 넘어선 예측 기반 경영 실현.

## 4. 전략적 가치 제안 (Strategic Value Proposition)
본 AX 대시보드는 단순한 시각화 도구를 넘어 다음과 같은 경영 가치를 제공합니다.

- **의사결정 속도 가속화 (Decision Velocity)**: 기존 수동 데이터 취합 방식(1~2일 소요)을 RPA 기반 실시간 동기화로 전환하여, 경영진이 '현재의 데이터'를 보고 즉시 판단할 수 있는 환경을 제공합니다.
- **최적 자원 배분 (Resource Optimization)**: AI 분석을 통해 특정 도크의 조업 지연이 예상될 경우, 유휴 인력을 적재적소에 재배치하는 가이드를 제공하여 인적 자원 효율을 극대화합니다.
- **인도 기일 준수 및 대외 신뢰도 (Contract Confidence)**: AI 기반의 완공 예정일(D-Day) 예측을 통해, 선주사(Owner)와의 인도 기일을 선제적으로 관리하고 조업 지연에 따른 지체보상금(LD) 리스크를 최소화합니다.

## 5. 데이터 획득 및 DB 연동 전략 (Data Acquisition)
- **예상 원천 데이터(Source)**: 한화오션 사내 ERP(SAP) 또는 조업 관리 DB(Oracle/MSSQL).
- **수집 방식**: 
    1. **RPA 하이브리드 수집**: DB 접근이 제한적인 경우 포털에서 직접 스크래핑.
    2. **ETL 파이프라인**: 전용 API나 DB Connector를 통해 정기적으로 CSV/Excel 형태로 Lakehouse에 적재.
- **활용**: 적재된 데이터를 바탕으로 실시간 공정 대시보드 및 Power BI 분석 리포트 자동 생성.
